# First Missing Positive

Given an unsorted integer array `nums`, return the smallest positive integer that is not present in `nums`.

Must run in **O(n) time** and use **O(1) auxiliary space**.

## Examples

- `[1,2,0]` → `3`
- `[3,4,-1,1]` → `2`
- `[7,8,9,11,12]` → `1`

## Constraints

- `1 <= nums.length <= 10^5`
- `-2^31 <= nums[i] <= 2^31 - 1`
